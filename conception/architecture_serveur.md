# Architecture serveur

## Objectif
Définir l’architecture du système serveur de l’infrastructure.

## Composants
- Active Directory
- DNS
- DHCP
- Serveur de fichiers

## Sécurité
- GPO de base
- Comptes administrateurs séparés
- Stratégies de mot de passe

## Résultat attendu
Infrastructure centralisée et sécurisée.
